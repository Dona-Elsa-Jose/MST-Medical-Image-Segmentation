# 🫁 MST-Medical-Image-Segmentation

An advanced graph-based segmentation framework for **Axial Lung CT imaging**. This project leverages the **Felzenszwalb-Huttenlocher algorithm** to perform precise anatomical isolation of the lung parenchyma, facilitating automated medical diagnostics.

## 📋 Requirements
To run this project, ensure you have the following installed:
* **Python 3.8+**
* **NumPy** (for array manipulations)
* **OpenCV** (for image I/O and preprocessing)
* **Matplotlib** (for result visualization)

## 🧠 Core Methodology
The system models the image as a 4-connected grid graph $G=(V, E)$, where pixels are nodes and edge weights represent intensity gradients.

### 1. Minimum Spanning Tree (MST) Construction
Using a Kruskal-inspired approach, we sort edges by weight and iteratively build components. This allows the algorithm to group pixels with similar intensity values while respecting anatomical boundaries.



### 2. DSU Optimization
To handle high-resolution medical data, we implemented a custom **Disjoint Set Union** with:
* **Path Compression:** Flattens the tree structure for $O(1)$ amortized lookups.
* **Union by Rank:** Keeps the tree balanced to prevent performance degradation.

### 3. Adaptive Thresholding
We use a dynamic predicate $MInt(C_1, C_2)$ that adjusts the merging sensitivity based on component size, preventing the "leaking" of lung segments into the chest wall.
